Switch Case

Pernyataan switch digunakan untuk menjalankan salah satu pernyataan dari beberapa kemungkinan pernyataan, berdasarkan nilai dari sebuah ungkapan atau kondisi. Pernyataan switch case dapat mengantikkan pernyataan If majemuk. Namun perlu diperhatikan beberapa syarat:
1. Casenya harus berupa alfabet atau numerik
2. Kondisi harus bernilai sebanding (==)

Bentuk umumnya sebagai berikut : 

    switch (ungkapan)
{
    case ungkapan1 :
        pernyataan_1 ;
	    break ;
    case ungkapan2 : 
        pernyataan_2 ;
	    break ;
	.......
	case ungkapanN :
        pernyataan_N ;
	    break ; 
    default :
	//opsional pernyataan-lain;
}
Penulisan case yang berupa alfabet menggunakan tanda petik satu. Contoh case 'a' untuk numerik tidak usah menggunakan tanda petik satu, comtoh case 1
