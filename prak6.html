JOBSHEET 6

Judul Praktikum: Array

Tujuan Praktikum

  • 1. Memahami konsep array dalam pemrogram C++ secara benar.
  • 2. Mengenal bentuk array dalam pemrograman baik berdimensi satu, dimensi dua atau dimensi banyak secara benar.
  • 3. Dapat menggunakan bentuk-bentuk array dalam aplikasinya pada pembuatan program secara tepat.
  • 4. Dapat mengembangkan bentuk-bentuk array dalam pemrograman secara benar.

Praktikum

Praktikum 6.1 (Array 1 Dimensi)

Deskripsi dan langkah-langkah praktikum while loop.


Praktikum 6.2 (Array 2 Dimensi)

Deskripsi dan langkah-langkah praktikum do while loop.


Praktikum 6.3 (Array Multidimensi)

Deskripsi dan langkah-langkah praktikum for loop.
