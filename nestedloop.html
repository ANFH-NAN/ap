Nested Loop (Perulangan Bersarang)

Nested Loop (perulangan bersarang) adalah sebuah struktur perulangan di mana sebuah loop berada di dalam loop lainnya atau bisa juga kita sebuah perulangan di dalam perulangan. Perulangan yang dikerjakan terlebih dahulu adalah perulangan yan berada di dalam (inner loop) baru kemudian perulangan yang luar (outer loop).
Berikut Animasi Bagaimana Cara Kerja Nested Loop.

#include <iostream> using namespace std; int main() { int i, j, tinggi; cout << "Masukkan tinggi segitiga: "; cin >> tinggi; for (i = 1; i <= tinggi; i++) { for (j = 1; j <= i; j++) { cout << "*"; } cout << endl; } return 0; }
